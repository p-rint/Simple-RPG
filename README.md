Made by Kenny in Godot
Game has a simple combat + animation system, innocent people you need to slime out, a shop for buying upgrades, and a main menu!
Run it by going to the [itch.io page](https://itchiback.itch.io/merk-innocent-civillians-for-money-rpg), and pressing "Run game"


Controls:
WASD - Move
Space - Jump
Tab - Unlock mouse
Click - Attack
R - To travel to shop (if you're in the mainland, you'll need to be near the pipe's entrance)
